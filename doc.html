Getting Started

D4rkzB is designed to be the simplest way to store and retrieve data. To start using D4rkzB, you first need an API Key from your Dashboard.

Installation

npm install d4rkzb

Usage Example

import { D4rkzB } from 'd4rkzb';

const db = new D4rkzB('YOUR_API_KEY');

// Insert data
await db.insert('users', { 
  name: 'John Doe', 
  email: 'john@example.com' 
});

// Fetch data
const users = await db.fetch('users');
