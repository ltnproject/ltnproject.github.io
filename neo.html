จากวินโดวส์ สู่ macOS ง่ายกว่าที่คิด

รวมทุกเรื่องที่คุณพ่อควรรู้ในสัปดาห์แรก ตั้งแต่การจัดการไฟล์ไปจนถึงเจสเจอร์บนแทร็คแพด แตะเลือกหัวข้อทางซ้าย แล้วค่อย ๆ ทำความคุ้นเคยได้เลยครับ ไม่ต้องรีบ

File Explorer→Finder

Finder คือหน้าต่างจัดการไฟล์ของ Mac เหมือนกับ File Explorer ที่คุณพ่อคุ้นเคยบนวินโดวส์เลยครับ ใช้ดูรูป เอกสาร และโฟลเดอร์ต่าง ๆ ได้ตามปกติ ไอคอนหน้ายิ้มสีฟ้าที่แถบด้านล่างสุดของจอ คือทางลัดเข้า Finder ได้ทันที

Control Panel→การตั้งค่าระบบ (System Settings)

ทุกอย่างที่เคยปรับใน Control Panel เช่น เสียง จอภาพ วันเวลา หรือการเชื่อมต่อ Wi-Fi จะย้ายมาอยู่ในเมนู "การตั้งค่าระบบ" แทน เปิดได้จากไอคอนรูปเฟืองในแถบ Dock ด้านล่าง ใช้งานง่าย จัดหมวดหมู่ชัดเจนกว่าเดิมด้วยครับ

Taskbar→Dock + แถบเมนูด้านบน

แถบไอคอนโปรแกรมด้านล่างจอเรียกว่า "Dock" ใช้เปิดแอปที่ใช้บ่อยได้เลย ส่วนแถบด้านบนสุดของจอคือ "แถบเมนู" จะแสดงเวลา สัญญาณ Wi-Fi และเมนูคำสั่งของแอปที่เปิดอยู่ ต่างจากวินโดวส์ที่รวมทุกอย่างไว้แถบเดียวด้านล่างครับ

แค่จำสองปุ่มหลักนี้ก็ใช้งานได้คล่องแล้วครับ ปุ่ม Command (⌘) จะอยู่ข้าง Spacebar ใกล้เคียงตำแหน่งปุ่ม Ctrl เดิมพอดี มือจะปรับตัวได้เร็วมาก

ปุ่มปิด ย่อ ขยายหน้าต่าง

บน Mac ปุ่มควบคุมหน้าต่างทั้งสามจะย้ายมาอยู่ "มุมบนซ้าย" แทนมุมขวาที่คุณพ่อคุ้นเคย โดยสีแดงคือปิดหน้าต่าง สีเหลืองคือย่อหน้าต่างเก็บไว้ใน Dock และสีเขียวคือขยายเต็มจอ ลองสังเกตสีแล้วจะจำได้ไม่ยากครับ

เจสเจอร์แทร็คแพดที่ใช้บ่อย

แทร็คแพดของ Mac ใช้นิ้วสัมผัสแทนการคลิกเมาส์ได้เกือบทุกอย่าง ลองฝึกไม้เหล่านี้ดูครับ จะทำให้ใช้งานคล่องขึ้นเยอะ

อยากหาไฟล์ เปิดแอป หรือคำนวณเลขง่าย ๆ ไม่ต้องไล่หาเอง กด ⌘ + Space ที่ไหนก็ได้ แล้วพิมพ์สิ่งที่ต้องการ Spotlight จะค้นหาให้ทันที ทั้งไฟล์ แอป และแม้แต่คำในอีเมล เร็วกว่าการไล่คลิกหาเยอะครับ

คลิกขวาบน Mac

แทร็คแพดของ Mac ไม่มีปุ่มขวาแยกเหมือนเมาส์ทั่วไป แต่คลิกขวาได้เหมือนเดิมด้วยการ "แตะ 2 นิ้วพร้อมกัน" บนแทร็คแพด จะขึ้นเมนูตัวเลือกให้เหมือนคลิกขวาบนวินโดวส์ทุกอย่าง ใช้ไม่กี่ครั้งก็คุ้นมือครับ

ติดตั้งโปรแกรมใหม่

แทนที่จะดาวน์โหลดไฟล์ .exe เหมือนวินโดวส์ บน Mac ส่วนใหญ่จะติดตั้งผ่าน "App Store" ซึ่งเป็นแอปสีฟ้าใน Dock เปิดขึ้นมา ค้นหาแอปที่ต้องการ แล้วกด "รับ" หรือ "ติดตั้ง" ได้เลย ปลอดภัยและไม่ต้องกังวลเรื่องไวรัสครับ

Print Screen→⌘ + Shift + ตัวเลข

ถ่ายภาพหน้าจอทั้งหมด กด ⌘ + Shift + 3 ส่วนถ้าอยากเลือกถ่ายเฉพาะบางส่วนของจอ ให้กด ⌘ + Shift + 4 แล้วลากเลือกพื้นที่ที่ต้องการ ภาพจะถูกบันทึกไว้ที่หน้าจอเดสก์ท็อปโดยอัตโนมัติ
